## 🔥 Key Highlights

### 🌑 Dark Mode Interface
- **Pure Black UI:** എഡിറ്റിംഗിൽ കൂടുതൽ ശ്രദ്ധ കിട്ടാൻ കറുത്ത ബാക്ക്ഗ്രൗണ്ട്.
- **Bottom Navigation:** മൊബൈലിൽ ഉപയോഗിക്കാൻ എളുപ്പമുള്ള ടൂൾ ബാർ.

### 🖼️ Pro Overlay & Layers
- CapCut-ൽ ഉള്ളത് പോലെ തന്നെ ഒന്നിന് മുകളിൽ ഒന്നായി ഫോട്ടോകൾ അടുക്കാം. 
- മൾട്ടിപ്പിൾ ലെയർ സപ്പോർട്ടും ലെയർ ട്രേയും ഉൾപ്പെടുത്തിയിട്ടുണ്ട്.

### 🧠 AI Background Remover
- ഒരു ക്ലിക്കിൽ ഫോട്ടോയിലെ ബാക്ക്ഗ്രൗണ്ട് നീക്കം ചെയ്യാം.
- ബ്രൗസറിൽ തന്നെ റണ്ണിംഗ് ആകുന്ന പവർഫുൾ AI.
